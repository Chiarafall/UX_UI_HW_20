Rogen Neilson House

Website redesign case study

About the Project

Our group proposed the re-designing of a not-for-profit organization’s webpage. We selected the redesign of Roger Neilson House, a palliative care facility in Ottawa, Ontario, Canada. Our initial evaluation was that the site required branding overhauls, UI assistance and readability.
Bootcamp group project

Problem

The problems we faced are mostly about readability and finding useful content about the organization itself. We put ourself in the shoes of a user trying to apply to a volunteer position and filling up forms.

Solution

  • Revising the navigation bar, adding a visible “about us” section to enhance clarity about the organization itself

  • Redesign the volunteer application process to make it easier for the user to apply

  • forms and possibility to save
